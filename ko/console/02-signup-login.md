# 콘솔 회원가입 & 로그인

{% hint style="info" %}
💡 bkend 콘솔에 접속하기 위한 회원가입과 로그인 방법을 안내합니다.
{% endhint %}

## 개요

bkend 콘솔은 소셜 로그인(Google, GitHub)과 매직 링크를 지원합니다. 별도의 비밀번호 설정 없이 간편하게 가입하고 로그인할 수 있습니다.

***

## 지원하는 로그인 방식

| 방식 | 설명 |
|------|------|
| **Google** | Google 계정으로 로그인 |
| **GitHub** | GitHub 계정으로 로그인 |
| **매직 링크** | 이메일로 일회성 로그인 링크 발송 |

***

## 회원가입하기

1. [https://console.bkend.ai](https://console.bkend.ai)에 접속하세요.
2. **회원가입** 페이지에서 로그인 방식을 선택하세요.

{% tabs %}
{% tab title="Google" %}
1. **Google로 계속하기** 버튼을 클릭하세요.
2. Google 계정을 선택하고 권한을 승인하세요.
3. 자동으로 콘솔 홈 화면으로 이동합니다.
{% endtab %}
{% tab title="GitHub" %}
1. **GitHub로 계속하기** 버튼을 클릭하세요.
2. GitHub 계정으로 로그인하고 권한을 승인하세요.
3. 자동으로 콘솔 홈 화면으로 이동합니다.
{% endtab %}
{% tab title="매직 링크" %}
1. 이메일 주소를 입력하세요.
2. **매직 링크 보내기** 버튼을 클릭하세요.
3. 받은 메일에서 로그인 링크를 클릭하세요.
4. 자동으로 콘솔 홈 화면으로 이동합니다.
{% endtab %}
{% endtabs %}

{% hint style="success" %}
✅ 최초 로그인 시 자동으로 계정이 생성됩니다. 별도의 회원가입 절차가 없습니다.
{% endhint %}

***

## 로그인하기

1. [https://console.bkend.ai](https://console.bkend.ai)에 접속하세요.
2. 회원가입 때 사용한 방식과 동일한 방식으로 로그인하세요.

{% hint style="warning" %}
⚠️ 매직 링크는 발송 후 제한된 시간 동안만 유효합니다. 만료된 경우 다시 요청하세요.
{% endhint %}

***

## 초대받은 경우

팀원이 콘솔에서 초대 메일을 보낸 경우:

1. 초대 메일에서 **수락** 링크를 클릭하세요.
2. 로그인 방식을 선택하여 인증하세요.
3. 해당 Organization에 자동으로 합류합니다.

***

## 에러가 발생했다면

| 증상 | 원인 | 해결 |
|------|------|------|
| Google 로그인 실패 | 팝업 차단 | 브라우저 팝업 허용 설정 |
| 매직 링크 메일 미수신 | 스팸 필터 | 스팸함 확인, 재발송 |
| 로그인 후 빈 화면 | 캐시 문제 | 브라우저 캐시 삭제 후 재시도 |
