# 커스텀 도메인

{% hint style="info" %}
💡 bkend 프로젝트에 커스텀 도메인을 설정하는 방법을 안내합니다.
{% endhint %}

## 개요

🚧 **이 문서는 준비 중입니다.**

커스텀 도메인을 설정하면 자체 도메인으로 bkend API에 접근할 수 있습니다. 도메인 등록, DNS 설정, SSL 인증서 발급에 대한 가이드를 준비하고 있습니다.

***

## 다룰 내용 (예정)

- 도메인 등록 절차
- DNS CNAME 설정
- 도메인 검증
- SSL 인증서 자동 발급
- 도메인 활성화 및 관리
