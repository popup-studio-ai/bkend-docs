# 콘솔에서 Storage 관리하기

> bkend 콘솔의 Storage 화면에서 파일을 관리하는 방법을 안내합니다.

## 개요

콘솔의 **Storage** 메뉴에서 업로드된 파일을 조회하고, 파일 메타데이터를 확인하며, 파일을 다운로드하거나 삭제할 수 있습니다.

---

## Storage 페이지 접근하기

1. 콘솔에서 프로젝트를 선택하세요.
2. Sidebar의 **Storage** 메뉴를 클릭하세요.
3. 상단의 환경 탭에서 원하는 환경(dev, staging, prod)을 선택하세요.

---

## 화면 구성

### 파일 목록

| 영역 | 설명 |
|------|------|
| **환경 탭** | dev, staging, prod 환경 전환 |
| **검색** | 파일명으로 검색 |
| **파일 목록** | 업로드된 파일 목록 (파일명, MIME 타입, 크기, 가시성, 업로드일) |
| **페이지네이션** | 페이지 이동 |

### 파일 상세

파일 목록에서 파일을 클릭하면 상세 정보를 확인할 수 있습니다:

| 항목 | 설명 |
|------|------|
| **파일명** | 원본 파일명 |
| **MIME 타입** | 파일 형식 (image/jpeg, application/pdf 등) |
| **크기** | 파일 크기 |
| **가시성** | public, private, protected, shared |
| **버킷** | 파일 카테고리 (images, documents 등) |
| **소유자** | 파일 업로드한 User |
| **업로드일** | 파일 생성 시점 |

### 저장소 통계

| 항목 | 설명 |
|------|------|
| **전체 파일 수** | 환경에 업로드된 총 파일 수 |
| **저장소 사용량** | 총 저장 용량 |

---

## 파일 다운로드하기

1. 파일 목록에서 다운로드할 파일을 선택하세요.
2. **다운로드** 버튼을 클릭하세요.
3. public 파일은 CDN URL로, private 파일은 임시 다운로드 URL로 다운로드됩니다.

---

## 파일 삭제하기

1. 파일 목록에서 삭제할 파일을 선택하세요.
2. **삭제** 버튼을 클릭하세요.
3. 확인 대화 상자에서 삭제를 확인하세요.

> ❌ **위험** - 파일 삭제는 되돌릴 수 없습니다. 삭제된 파일은 복구할 수 없습니다.
