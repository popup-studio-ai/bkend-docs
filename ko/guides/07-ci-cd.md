# CI/CD 파이프라인

{% hint style="warning" %}
🚧 이 문서는 준비 중입니다. bkend 환경(dev/staging/prod)을 활용한 CI/CD 구성 가이드는 향후 제공될 예정입니다.
{% endhint %}

## 개요

bkend의 환경 분리(dev/staging/prod)를 앱의 CI/CD 파이프라인에 연동하는 가이드를 준비하고 있습니다.

***

## 준비 중인 내용

- 환경별 API 키 관리 (GitHub Secrets, Vercel 환경 변수)
- GitHub Actions에서 bkend 환경 활용
- 배포 플랫폼별 환경 변수 설정 (Vercel, Netlify 등)
- 환경별 테스트 자동화
