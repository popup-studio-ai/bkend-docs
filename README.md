# Welcome

**bkend**는 AI 도구에서 바로 사용할 수 있는 백엔드 서비스입니다.

MCP(Model Context Protocol)를 통해 Claude Code, Cursor 등 AI 도구에서 Database, Authentication, Storage 기능을 즉시 사용할 수 있습니다.
