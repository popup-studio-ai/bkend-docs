# 웹훅

{% hint style="info" %}
💡 bkend의 웹훅 기능을 설정하고 이벤트를 수신하는 방법을 안내합니다.
{% endhint %}

## 개요

🚧 **이 문서는 준비 중입니다.**

웹훅을 사용하면 bkend에서 발생하는 이벤트를 외부 서버로 실시간 전달할 수 있습니다. 데이터 변경, 사용자 가입 등의 이벤트를 활용한 자동화 가이드를 준비하고 있습니다.

***

## 다룰 내용 (예정)

- 웹훅 이벤트 타입
- 페이로드 형식
- 수신 서버 구현 (Node.js)
- 재시도 정책
- 멱등성 보장 패턴

{% hint style="warning" %}
⚠️ 웹훅 수신 서버는 **HTTPS** 엔드포인트여야 하며, 반드시 **서명 검증**을 구현하여 위조된 요청을 차단하세요.
{% endhint %}
