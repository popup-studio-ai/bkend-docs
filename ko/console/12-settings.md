# 프로젝트 설정

{% hint style="info" %}
💡 프로젝트의 기본 정보와 설정을 관리하는 방법을 안내합니다.
{% endhint %}

## 개요

프로젝트 설정 페이지에서 Project ID 확인, 프로젝트 정보 수정, 인증 설정, 프로젝트 삭제 등을 수행합니다.

***

## 설정 페이지 열기

프로젝트 레벨 사이드바에서 **설정**을 클릭하세요.

***

## Project ID 확인하기

프로젝트 설정 페이지 상단에 **Project ID**가 표시됩니다. REST API 호출 시 `X-Project-Id` 헤더에 이 값을 사용합니다.

```bash
-H "X-Project-Id: {project_id}"
```

{% hint style="info" %}
💡 Project ID는 자동 생성된 고유 식별자입니다. 변경할 수 없습니다.
{% endhint %}

***

## 프로젝트 정보 수정하기

| 필드 | 수정 가능 | 설명 |
|------|:--------:|------|
| **프로젝트 이름** | ✅ | 표시 이름 변경 |
| **프로젝트 Slug** | ✅ | URL 식별자 변경 |
| **리전** | ❌ | 생성 후 변경 불가 |
| **Project ID** | ❌ | 자동 생성, 변경 불가 |

***

## 인증 제공자 설정

프로젝트 설정에서 User 인증에 사용할 제공자를 활성화/비활성화합니다.

- [인증 제공자 설정](../authentication/17-provider-config.md) — 상세 설정 방법

***

## 프로젝트 삭제하기

{% hint style="danger" %}
🚨 **위험** — 프로젝트를 삭제하면 모든 환경, 테이블, User, 파일, API Key가 영구 삭제됩니다. 이 작업은 되돌릴 수 없습니다.
{% endhint %}

1. 설정 페이지 하단의 **프로젝트 삭제** 섹션으로 이동하세요.
2. **삭제** 버튼을 클릭하세요.
3. 프로젝트 이름을 입력하고 **삭제 확인**을 클릭하세요.
