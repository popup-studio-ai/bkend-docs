# 대시보드

{% hint style="info" %}
💡 대시보드에서 프로젝트의 현황을 한눈에 파악하세요.
{% endhint %}

## 개요

bkend 콘솔은 조직 레벨과 프로젝트 레벨에서 각각 대시보드를 제공합니다. 주요 통계, 최근 활동, 빠른 액션을 통해 현재 상태를 빠르게 파악할 수 있습니다.

***

## 조직 대시보드

사이드바에서 **홈**을 클릭하면 조직 대시보드가 표시됩니다.

### 빠른 통계

| 항목 | 설명 |
|------|------|
| 프로젝트 수 | 조직 내 전체 프로젝트 수 |
| 팀 멤버 수 | 활성 멤버 수 |
| 스토리지 사용량 | 전체 파일 스토리지 사용량 |
| 테이블 수 | 전체 프로젝트의 테이블 총 수 |

### 최근 프로젝트

최대 6개의 프로젝트가 카드 형태로 표시됩니다. 각 카드에는 프로젝트명, 설명, 리전, 클라우드 정보가 포함됩니다.

### 최근 활동

조직 내에서 발생한 최근 활동이 시간순으로 표시됩니다. 프로젝트 생성, 환경 변경, 멤버 초대 등의 이벤트를 확인할 수 있습니다.

## 프로젝트 대시보드

프로젝트를 선택하면 프로젝트 대시보드가 표시됩니다.

### 빠른 통계

| 항목 | 설명 |
|------|------|
| Database | 테이블 수 |
| Storage | 스토리지 사용량 |
| Authentication | 등록된 User 수 |

### 프로젝트 정보

| 항목 | 설명 |
|------|------|
| Project ID | 고유 식별자 (복사 가능) |
| Region | 데이터 저장 리전 |
| Cloud | 클라우드 제공자 |
| 생성 날짜 | 프로젝트 생성일 |
| Environments | 환경 수 |

### 빠른 액션

- **Database 관리** — 테이블 목록으로 이동
- **Authentication 관리** — 인증 설정으로 이동
- **File Storage 관리** — 스토리지로 이동
- **Environments 관리** — 환경 목록으로 이동

### 최근 활동

프로젝트 내에서 발생한 최근 활동이 시간순으로 표시됩니다. **View All** 링크를 클릭하면 전체 활동 로그를 확인할 수 있습니다.
