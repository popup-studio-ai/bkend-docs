# 콘솔에서 API Key 관리하기

> 콘솔에서 API Key를 생성하고 관리하는 방법을 안내합니다.

## 개요

콘솔의 Organization 설정에서 API Key를 생성, 조회, 폐기할 수 있습니다. Owner 또는 Admin 역할이 필요합니다.

---

## API Key 페이지 접근하기

1. 콘솔 상단에서 Organization을 선택하세요.
2. **설정** 메뉴로 이동하세요.
3. API Key 관련 설정을 확인하세요.

---

## API Key 생성하기

1. **API Key 생성** 버튼을 클릭하세요.
2. **이름**을 입력하세요 (예: "Production API Key").
3. **타입**을 선택하세요 (API Key 또는 Bearer Token).
4. **권한 범위(Scopes)**를 선택하세요.
5. 필요한 경우 **만료 시간**을 설정하세요.
6. **생성**을 클릭하세요.

> ❌ **위험** - 생성된 API Key는 이 시점에만 확인할 수 있습니다. 반드시 안전한 곳에 복사하여 저장하세요.

---

## API Key 목록

| 항목 | 설명 |
|------|------|
| **이름** | API Key 이름 |
| **타입** | API_KEY 또는 BEARER_TOKEN |
| **마스킹된 키** | `ak_8930****...dcbb` 형태 |
| **스코프** | 부여된 권한 목록 |
| **생성자** | 생성한 Tenant 이름 |
| **생성일** | 생성 시점 |
| **만료일** | 만료 시점 (설정된 경우) |
| **상태** | 활성 / 폐기됨 |

---

## API Key 폐기하기

1. API Key 목록에서 폐기할 키를 선택하세요.
2. **폐기** 버튼을 클릭하세요.
3. 확인 대화 상자에서 폐기를 확인하세요.

> ⚠️ **주의** - 폐기된 API Key는 즉시 무효화됩니다. 해당 키를 사용하는 모든 서비스가 영향을 받습니다.
