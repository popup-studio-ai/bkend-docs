# 대시보드

{% hint style="info" %}
💡 대시보드는 콘솔 로그인 후 표시되는 랜딩 페이지입니다. 마지막으로 사용한 조직과 프로젝트로 자동 이동합니다.
{% endhint %}

## 개요

콘솔에 접속하면 현재 컨텍스트에 따라 대시보드가 자동으로 리다이렉트합니다.

***

## 리다이렉트 동작

| 컨텍스트 | 이동 대상 |
|---------|----------|
| 조직과 프로젝트가 선택된 상태 | 프로젝트 개요 페이지 |
| 조직만 선택된 상태 | 조직 홈 페이지 |
| 조직이 없는 상태 | **조직 생성** 안내가 표시된 콘솔 홈 |

{% hint style="info" %}
💡 이전에 조직과 프로젝트를 선택한 적이 있으면, 콘솔이 선택 상태를 기억하고 바로 이동합니다.
{% endhint %}

***

## 프로젝트 개요

프로젝트를 선택한 후, 프로젝트 레벨 사이드바에서 **개요**를 클릭하면 프로젝트 랜딩 페이지를 확인할 수 있습니다.
