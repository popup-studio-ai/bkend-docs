# 대시보드

{% hint style="info" %}
💡 프로젝트 대시보드에서 주요 통계와 상태를 한눈에 확인합니다.
{% endhint %}

## 개요

프로젝트를 선택하면 첫 화면으로 대시보드가 표시됩니다. 프로젝트의 주요 리소스 현황과 활동 상태를 확인할 수 있습니다.

***

## 대시보드 확인하기

프로젝트 레벨 사이드바에서 **개요**를 클릭하세요.

***

## 표시 정보

| 항목 | 설명 |
|------|------|
| **환경** | 환경 수 및 상태 |
| **데이터베이스** | 현재 환경의 테이블 개수 |
| **스토리지** | 스토리지 사용량 |
| **인증** | 등록된 사용자 수 |
| **API 통계** | 이번 달 API 호출 통계 |

***

## 빠른 액션

대시보드에서 자주 사용하는 기능으로 빠르게 이동할 수 있습니다.

| 액션 | 설명 |
|------|------|
| **데이터베이스** | 테이블 관리로 이동 |
| **인증** | 사용자 및 인증 제공자 관리로 이동 |
| **파일 스토리지** | 파일 업로드 및 관리로 이동 |
| **환경** | 환경 관리로 이동 |

***

## 최근 활동

대시보드 하단에 프로젝트에서 수행된 최근 활동이 표시됩니다. **전체 보기**를 클릭하면 활동 로그 전체 목록으로 이동합니다.

{% hint style="info" %}
💡 대시보드에 표시되는 데이터는 현재 선택된 환경 기준입니다. 환경을 전환하면 해당 환경의 통계가 표시됩니다.
{% endhint %}
