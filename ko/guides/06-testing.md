# 테스트 전략

{% hint style="info" %}
💡 bkend 프로젝트의 테스트 환경 구성과 테스트 전략을 안내합니다.
{% endhint %}

## 개요

🚧 **이 문서는 준비 중입니다.**

bkend의 환경 분리(dev/staging/prod) 기능을 활용하면 안전하게 테스트를 수행할 수 있습니다. CRUD, 인증, 권한 테스트 전략에 대한 상세 가이드를 준비하고 있습니다.

***

## 다룰 내용 (예정)

- 환경별 테스트 전략 (dev, staging, prod)
- CRUD 테스트 코드 예시
- 인증 테스트 (회원가입, 로그인, 토큰 갱신)
- RLS 권한 테스트 시나리오
- 테스트 자동화 스크립트
