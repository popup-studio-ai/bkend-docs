# 테이블 관리

{% hint style="info" %}
💡 콘솔에서 테이블을 생성하고 데이터를 관리하는 방법을 안내합니다. 테이블 생성은 콘솔 또는 MCP 도구로만 가능합니다.
{% endhint %}

## 개요

데이터베이스 메뉴에서 테이블을 생성, 조회, 삭제합니다. 테이블 생성/관리는 콘솔 UI 또는 [MCP 테이블 관리 도구](../mcp/04-table-tools.md)를 사용합니다. REST API로는 데이터 CRUD만 가능합니다.

***

## 테이블 목록 확인하기

1. 프로젝트 레벨 사이드바에서 **데이터베이스**를 클릭하세요.
2. 현재 환경의 모든 테이블 목록을 확인합니다.

| 표시 정보 | 설명 |
|----------|------|
| **테이블 이름** | 테이블 식별자 |
| **컬럼 수** | 필드 개수 |
| **데이터 수** | 저장된 레코드 수 |

***

## 테이블 생성하기

{% tabs %}
{% tab title="콘솔" %}
1. **테이블 생성** 버튼을 클릭하세요.
2. 테이블 이름을 입력하세요 (예: `posts`).
3. **생성**을 클릭하세요.
4. [스키마 편집기](08-schema-editor.md)에서 컬럼을 추가하세요.
{% endtab %}
{% tab title="MCP" %}
AI 도구에서 자연어로 요청하세요.

```text
"posts 테이블을 만들어줘.
- title: 문자열 (필수)
- content: 문자열 (필수)
- published: 불리언 (기본값: false)"
```

MCP 도구가 자동으로 테이블을 생성하고 컬럼을 추가합니다.
{% endtab %}
{% endtabs %}

{% hint style="info" %}
💡 테이블을 생성하면 `id`, `createdAt`, `updatedAt` 필드가 자동으로 추가됩니다.
{% endhint %}

***

## 테이블 데이터 조회하기

1. 테이블 목록에서 테이블을 클릭하세요.
2. 저장된 데이터를 테이블 형태로 확인합니다.
3. 검색, 필터링, 페이지 이동을 사용할 수 있습니다.

***

## 테이블 삭제하기

{% hint style="danger" %}
🚨 **위험** — 테이블을 삭제하면 모든 데이터가 영구 삭제됩니다. 이 작업은 되돌릴 수 없습니다.
{% endhint %}

1. 테이블 목록에서 삭제할 테이블의 메뉴를 클릭하세요.
2. **삭제**를 선택하세요.
3. 테이블 이름을 입력하고 **삭제 확인**을 클릭하세요.
