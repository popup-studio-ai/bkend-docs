# 대시보드

{% hint style="info" %}
💡 프로젝트 대시보드에서 주요 통계와 상태를 한눈에 확인합니다.
{% endhint %}

## 개요

프로젝트를 선택하면 첫 화면으로 대시보드가 표시됩니다. 프로젝트의 주요 리소스 현황과 활동 상태를 확인할 수 있습니다.

***

## 대시보드 확인하기

프로젝트 레벨 사이드바에서 **개요**를 클릭하세요.

***

## 표시 정보

| 항목 | 설명 |
|------|------|
| **환경 목록** | 프로젝트의 환경 현황 (Active/Inactive) |
| **테이블 수** | 현재 환경의 테이블 개수 |
| **User 수** | 현재 환경의 등록 사용자 수 |
| **파일 수** | 현재 환경의 저장 파일 수 |
| **Project ID** | REST API에서 사용하는 프로젝트 식별자 |

{% hint style="info" %}
💡 대시보드에 표시되는 데이터는 현재 선택된 환경 기준입니다. 환경을 전환하면 해당 환경의 통계가 표시됩니다.
{% endhint %}
